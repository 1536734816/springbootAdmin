# springbootAdmin

前端基于Vue搭建

#####2.用户管理（开发完成）
#####3.角色管理（开发中）
#####4.权限管理（未完成）
#####5.用户角色分配（未完成）
#####6.角色权限分配（未完成）

#####8.角色CURD操作（未完成）
#####9.权限CURD操作（未完成）
